The subfilter module.¶

This module implements overall filtering.

The main functions are:

  • subfilter.subfilter.filter_variable_list().

  • subfilter.subfilter.quadratic_subfilter().

Before use, two files have to be created, using:

  • subfilter.subfilter.setup_derived_data_file().

  • subfilter.subfilter.setup_filtered_data_file().

New at 0.3

  1. The subfilter.subfilter.filter_variable_pair_list() function outputs filtered pairs inder the name “s()_on_grid” where “grid” will be “u”, “v”, “w” or “p”.
