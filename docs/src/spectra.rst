=====================
The spectra module.
=====================
This module implements computation of 1D spectra from 2D fields. 
It is a new addition at 0.5.0.

The main functions are:

* :py:func:`subfilter.spectra.spectrum_ave_1D`.
* :py:func:`subfilter.spectra.spectrum_ave_1D_radial`.
